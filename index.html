HTML5 CHEATSHEET

html basic tags

Example: <p>insert paragraph text here<p>

tag name description
<!DOCTYPE html> !doctype all html documents must start with a document type declaration.
<html> html the document begins with the html tag where the content is inside.
<body> body the visible part of the html document.
<header> header the visible header of the html document.
<footer> footer the visible footer of the html document.
<div> div defines a division/section in an html document.
<h1-h6> headings defines the html headings and there is 6 levels of hierarchy in total.
<p> paragraph defines a html paragraph.
<a> anchor point defines a hyperlink on the given text. must be used in conjunction with an attribute.
<img> image displays an image in conjunction with an attribute.
<!--comment--> comment creates comment made by the developer. (can only be seen in source code)

html attributes

Example: <img src="img_girl.jpg" width="500" height="600">

tag name description
href="" href defines the hyperlink url of the page that the link goes to.
src="" src specifies the path of the image to be displayed.
width="" & height="" width and height specifies the width and height of the image (in pixels).
alt="" alt if the image for some reason cannot be displayed. descriptive text will be shown instead.
style="" style used to add styles to an element, such as color, font, size and more.
lang="" language used with html tag. declares the language of the webpage. seo optimisation.
title="" title defines extra information about and element.

HTML Formatting Elements

Example: <b>bold text here</b>

tag name description
<b> bold bolds text
<strong> strong important text
<i> italic italic text
<em> emphasize emphasizes text
<mark> mark marked text
<small> small smaller text
<del> del deleted text
<ins> inserted inserted text
<sub> subscript subscripts text
<sup> superscript superscripts text

table tags

tag name description
<table> table the wrapper element for all html tables.
<thead> table head the set of rows defining the column headers in a table.
<tbody> table body the set of rows containing actual table data.
<tr> table row the table row container.
<td> table data defines a standard data cell in an html table.
<tfoot> table foot the set of rows defining the footer in a table.

table attributes

attribute name description
colspan column span defines how many columns a td element should span.
rowspan row span defines how many rows a td element should span.

form tags

tag name description
<form> form the element for all html forms.
<input> input the element for allowing user inputs.
<label> label defines a lable for form elements.

form <input> attributes

tag name description
<input type="text"> input type text displays a single-line text input field.
<input type="radio"> input type radio displays a radio button (for selecting one of many choices).
<input type="checkbox"> input type checkbox displays a checkbox (for selecting zero or more of many choices).
<input type="submit"> input type submit displays a submit button (for submitting the form).
<input type="button"> input type button displays a clickable button.
